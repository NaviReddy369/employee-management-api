Employee API — local UI

Open this page at http://127.0.0.1:8000/ (same origin) so the buttons can call the API.

How to run

  1. In the project folder, start the stack: docker compose up --build
  2. Wait until the API log shows Application startup complete.
  3. Open http://127.0.0.1:8000/ in your browser (this page).

If you changed the database schema, reset the DB volume once: docker compose down -v then docker compose up --build.
